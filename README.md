# My React Boilerplate

## Install
```
git clone https://github.com/yuruga/react-boilerplate.git
```
or [download zip](https://github.com/yuruga/react-boilerplate/archive/master.zip)

## Package Install

```
npm install
```

## Test

```
npm test
```

## Local Server

```
npm start
```

## Build

```
npm run build
```
